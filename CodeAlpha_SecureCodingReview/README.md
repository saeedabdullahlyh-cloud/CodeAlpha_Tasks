# 🔐 Secure Coding Review

## CodeAlpha Cyber Security Internship - Task 3

### 📖 Project Description

This project was developed as part of the **CodeAlpha Cyber Security Internship**.

The objective of this project is to perform a secure coding review of a Python-based login application by identifying common security vulnerabilities and demonstrating how they can be mitigated using secure coding practices.

Two implementations are included:

- Vulnerable Login System
- Secure Login System

The project highlights insecure coding practices and provides a secure implementation using password hashing with **bcrypt**.

---

## 🎯 Objectives

- Perform secure code review
- Identify security vulnerabilities
- Demonstrate insecure coding practices
- Implement secure coding techniques
- Compare vulnerable and secure implementations

---

## 🔍 Vulnerabilities Identified

- Hardcoded Credentials
- Plain Text Password Storage
- Plain Text Password Verification
- Weak Authentication
- Limited Input Validation

---

## ✅ Security Improvements

- Password Hashing using bcrypt
- Secure Password Verification
- Generic Error Messages
- Improved Authentication
- Better Secure Coding Practices

---

## 🛠️ Technologies Used

- Python 3
- bcrypt
- Visual Studio Code
